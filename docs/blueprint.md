# **App Name**: CruiseLink

## Core Features:

- User Authentication: Secure login/registration for voyagers and admin using Firebase authentication.
- Catering Orders: Voyagers can order snacks, food, and beverages.
- Stationery Orders: Voyagers can order gift items, chocolates, and tale books.
- Ticket Booking: Voyagers can book tickets for Resort-Movies, with seating availability checks.
- Service Bookings: Voyagers can book Beauty Salon, Fitness Center, and Party Hall services.
- Admin Dashboard: Admin to manage items, menus, and voyager registrations.
- Report Generation: Generative AI tool that analyses all voyage events to find any inefficiencies or bottlenecks for future trip improvements.

## Style Guidelines:

- Primary color: Deep blue (#1A237E) to evoke a sense of trust and the sea.
- Background color: Very light blue-gray (#F0F4F7) to provide a clean, calm backdrop.
- Accent color: Soft orange (#FF7043) for interactive elements, to suggest adventure.
- Body and headline font: 'Inter' (sans-serif) for a modern and readable interface.
- Code font: 'Source Code Pro' for displaying code snippets.
- Simple, modern icons to represent each service category.
- Subtle transitions and animations to enhance user experience when booking services.
